# User Information Form

A simple web form that collects user information and displays it in a formatted message.

## Features

- Input fields for name and age
- Checkboxes to select programming skills (Python and JavaScript)
- Displays formatted output message when form is submitted
- Simple, minimal styling

## How to Use

1. Open `index.html` in a web browser
2. Enter your name and age
3. Select the programming skills you know
4. Click Submit
5. Your information will be displayed below the form in the format: "My name is [name] and I am [age] years old and can code in [skills]."

- `index.html` - HTML form structure
- `index.css` - Basic styling
